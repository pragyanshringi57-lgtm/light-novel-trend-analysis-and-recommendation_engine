# Light Novel Recommendation System 📚

An intelligent, NLP-powered recommendation engine designed to suggest light novels based on storyline, themes, and user preferences. 

Finding the next great read can be overwhelming with thousands of titles out there. This project solves that by moving beyond simple genre tags and utilizing Natural Language Processing (NLP) to analyze the actual synopses of light novels. By computing the semantic similarity between story descriptions using word embeddings, the system recommends novels that truly capture the vibe of your favorites.

---

## 🚀 Features

* **Content-Based Filtering:** Recommends new novels by analyzing the text of the synopses rather than relying solely on user ratings.
* **Advanced NLP Processing:** Cleans, tokenizes, and processes raw text data from light novel databases.
* **Word Embeddings:** Utilizes word embedding models to capture the deep semantic meaning of story descriptions.
* **Personalized Output:** Simply input a title like *Mushoku Tensei* or *Classroom of the Elite*, and the system will output the top recommended titles with the highest similarity scores.
